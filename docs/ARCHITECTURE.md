Esse é um projeto estático simples, rodando em React, com TypeScript, TailwindCSS e Vite.

Ele é composto por uma página, que é dividida em seções. Cada seção é um componente React, que é responsável por renderizar a seção.

Mais informação sobre seções e componentes está em `PRD.md`.

O projeto é responsivo e funciona em diferentes tamanhos de tela.
