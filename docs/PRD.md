# PRD — Portfolio Website

## Proposta

Será um site para mostrar meu portfólio, com informações sobre mim, minhas habilidades, projetos e experiências. O site será responsivo e funcionará em diferentes tamanhos de tela. Será um site single page, em uma unica tela, com scroll para as seções. Na seção de projetos e experiências, será possível clicar em um botão para ver mais informações sobre o projeto ou experiência, nesse caso irá abrir uma página de detalhe para mostrar detalhes do projeto, como descrição, tecnologias, imagens, vídeos e link para o projeto.

O site deve ter duas versões, em português e em inglês. O usuário poderá alternar entre as duas versões. Escreva tudo originalmente em inglês e depois crie a versão em português e traduza para o português.

## Seções

- Header
- Hero Section
- About
- Skills
- Projects
- Experience
- Contact
- Resume
- Footer

# Header

O header deve conter:

- Logo, ficará em assets/images/logo.png
- Links para as seções
- O ultimo botão é um download do currículo, que ficará em assets/docs/resume.pdf

# Hero Section

O hero deve conter:

- Foto, ficará em assets/images/profile.png
- Título
- Texto (titulo, subtitulo e descrição)
- Botão do Linkedin e Github

# About

O about deve conter:

- Titulo
- Descrição
- Histórico Acadêmico

# Skills

- Tecnologias que eu uso, separado por categorias (exemplo: Backend, Frontend, AI, DevOps, etc)
- Seção extra de competências principais.

# Projects

- Lista de projetos que eu já fiz, com descrição, tecnologias usadas e link para o projeto.

# Experience

- Lista de experiências profissionais que eu já tive, com descrição, tecnologias usadas e link para o projeto.

# Contact

- Formulário de contato, com campos de nome, e-mail e mensagem.
- Botão de enviar mensagem.

# Resume

- Download do currículo, que ficará em assets/docs/resume.pdf

# Footer

- Informações de contato
- Links para redes sociais
- Copyright
